# ✋ Fine-Tuned Hand Detection Model

A real-time computer vision application designed to detect, track, and analyze human hand structures and gestures. This project was developed as a collaborative group effort, with the dataset customization, transfer learning strategy, and model fine-tuning process executed.

## ✨ Key Features
- Hand Tracking: High-accuracy landmark detection 
- Fine-Tuned Pipeline: Leveraged transfer learning on a pre-trained architecture to optimize accuracy for a specialized dataset.
- Real-Time Inference: Fast execution loop designed for live video tracking and camera feeds.

## 🛠️ Tech Stack
- Language: Python
- Approach: Transfer Learning & Fine-Tuning
- Libraries: OpenCV, MediaPipe / TensorFlow
